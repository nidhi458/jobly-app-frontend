# Jobly Frontend

This repository contains the frontend for the Jobly Job Application Platform. It provides user and admin interfaces for managing job listings and applications.

---

## Live Application
https://jobly-hub.netlify.app

---

## Backend API
https://jobly-0g1m.onrender.com/api

---

## Features

### User
- Register and login
- View job listings
- Apply for jobs
- Withdraw applications
- Filter jobs by company, location, and type

### Admin
- Create job listings
- Edit job details
- Delete jobs
- View all jobs in dashboard

---

## Tech Stack
- React
- Material UI
- Redux Toolkit (RTK Query)
- React Router

---

## Setup Instructions

### 1. Clone the repository

git clone https://github.com/YOUR_USERNAME/job-app-frontend
cd job-app-frontend

### 2. Install dependencies

npm install

### 3. Configure Environment Variables

REACT_APP_API_URL=http://localhost:5000/api

### 4. Run the application

npm start

---
### Deployment 
The frontend is deployed on Netlify.
